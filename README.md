# Курс «Корректность программ и операционные системы»

Курс посвящён организации программы в рамках операционной системы, вопросу организации эффективной и надёжной коммуникации между программами в рамках компьютерной сети, а также вопросу практической верификации программ.

В качестве объекта рассмотрения выбрана ОС Linux.

В курсе рассмотрены вопросы:
- Организация многопоточных вычислений.
- Блокирующая синхронизация между подпрограммами.
- Неблокирующая синхронизация между подпрограммами, простейший lock-free алгоритм.
- Синхронные и асинхронные интерфейсы ввода-вывода.
- Мультиплексирование потоков данных.
- Организация сетевого взаимодействия: модели UDP/IP, TCP/IP.
- Применение инструментов верификации ПО.

Читается 3 курсу ВМК МГУ на базе ИСП РАН.
