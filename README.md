# Курс «Операционные системы и компьютерные сети»

Курс посвящён организации ввода-вывода программы в рамках операционной системы, а также вопросу организации эффективной и надёжной коммуникации между программами в рамках компьютерной сети.

В качестве объекта рассмотрения выбрана ОС Linux.

В курсе рассмотрены вопросы:
- Организация многопоточных вычислений, синхронизация между подпрограммами.
- Синхронные и асинхронные интерфейсы ввода-вывода.
- Неблокирующая синхронизация.
- Мультиплексирование потоков данных (ToDo).
- Организация сетевого взаимодействия: модели UDP/IP, TCP/IP (ToDo).
- Архитектура драйверов устройств в ОС.

Читается 3 курсу ВМК МГУ на базе ИСП РАН.
